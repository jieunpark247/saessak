### 기능 및 페이지

- firebase 와 연동하여 SNS 로그인 페이지 구현 

- firebase realtime database, storage를 통해 바코드 인식 및 사진 저장 기능 구현 

- 각 지역구별 랭킹 확인 기능 구현

- 공공 API를 활용한 환경정보, 페트병재활용 정보 수질 등 확인 가능
